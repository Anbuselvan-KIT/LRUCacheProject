# LRU Cache in Java

This is my Java implementation of an LRU (Least Recently Used) Cache using a HashMap and a Doubly Linked List.

## Features
- Put key-value pairs
- Get values by key
- Automatically removes the least recently used item when the cache is full
- Console-based menu

## Technologies
- Java
- HashMap
- Doubly Linked List

## How to Run
1. Compile the Java files.
2. Run `Main.java`.
3. Enter the cache capacity and use the menu options.

## What I Learned
- How LRU Cache works
- Combining HashMap with a Doubly Linked List
- Managing pointers and cache operations in O(1) time
